# Changelog

## 0.1.0

- Initial release.
- Template derived from the LaTeX class `algotel.cls`.
- French and English support (`lang: "fr"` / `lang: "en"`).
- Theorem environments via lemmify: `theorem`, `lemma`, `proposition`, `corollary`, `definition`, `remark`, `example`, `proof`.
- Font fallback chain: Libertinus Serif, TeX Gyre Termes, Liberation Serif (body); TeX Gyre Heros, Liberation Sans (headings); TeX Gyre Cursor, Liberation Mono (code).
- Configurable front matter spacing.
- Alphanumeric citation style.
- Sample documents in French and English.
